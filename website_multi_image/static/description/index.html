Website Multi Images

Add product images in Odoo!!

Odoo Website Product Multi Images allows you to set multiple images for the products available on odoo website. Your website customers can easily view the images, they can simply go back and forth and can also use the slider to view all images. Multiple images are displayed in a slider and you can choose an image from the slider.
