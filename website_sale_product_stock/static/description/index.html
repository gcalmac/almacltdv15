Odoo Website Product Stock Items Odoo Apps

In odoo allow to show In stock product items and out of stock items on ecommerce page on Odoo website, This Odoo apps/modules also display stock in Webshop interface also the seller can select and display different type of stock like "Qty on hand" and "Qty Available" in Website Configuration from back-end and, also Enable/disable stock messages and validation from the webshop in Odoo ERP.

Features

Display Quantity Status

Seller can select and display different type of stock like "Qty on hand" and "Qty Available" in Website Configuration from back-end.

Hide/Show Stock Quantity

In website you can hide/show products stock quantity. Enable/disable stock messages and validation from the Webshop.

Remainder Mail

We can set up timing for email reminder via cron about availability of stock.

Stock Type Configuration

If you select stock type configuration as Qty On Hand then it display On hand Qty of product.

Show Product Availability

Show product availability for visitor and register customer. Show product availability for different product variants too.

Configure Notification

Allow customer/visitor set up notification for interested product not available in stock.

Notification for In/Out Stock

Its shows customer message for In Stock and Out Of Stock product. Its shows "In Stock" and "Out Of Stock" Message for different product variants too.

Enable/Disable Messages and Validation Webshop

Enable/disable stock messages and validation from the webshop.

Product Configuration : Sell Regardless of Inventory

You can add to cart the product and process the order in ecommerce although product doesn't have any onhand qty.

Product doesn't have onhand qty, than also process checkout and validated

when you select sell regardless of inventory option that time product and process the order in ecommerce although product doesn't have any onhand qty.
